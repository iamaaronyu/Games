# 🐍 贪吃蛇 — 扩展计划

## 优先级说明

- 🔴 **P0**：核心体验提升，优先实现
- 🟡 **P1**：丰富玩法，第二批次
- 🟢 **P2**：锦上添花，可选

---

## 🔴 P0 — 核心体验

### 1. 金苹果（特殊道具）

已规划，待实现。

- 地图上随机出现金色苹果，+3 分
- 出现后 5 秒自动消失（闪烁提示即将消失）
- 每吃 3~8 个普通苹果后触发出现

### 2. 移动端支持

当前只支持键盘操作，移动端无法游玩。

- 添加虚拟方向键（canvas 下方）
- 支持**滑动手势**控制方向
- 响应式画布尺寸

### 3. 难度选择

开局前选择难度，影响初始速度和速度增长曲线。

| 难度 | 初始速度 | 增速 | 上限 |
|------|----------|------|------|
| 简单 | 250ms | 3ms | 80ms |
| 普通 | 200ms | 5ms | 60ms |
| 困难 | 150ms | 7ms | 45ms |

---

## 🟡 P1 — 玩法丰富

### 4. 障碍物系统

地图上周期性生成障碍物（石头/墙壁），撞到即死。

- 每吃 5 个食物生成 1 个障碍物
- 障碍物不覆盖蛇身和食物
- 障碍物数量有上限（如最多 8 个）

### 5. 穿墙模式

吃到特殊道具后，蛇可以穿过墙壁从另一侧出现。

- 道具持续 8 秒
- 期间蛇身为霓虹色闪烁

### 6. 关卡系统

- 每吃 10 个食物进入下一关
- 每关地图变化：障碍物布局不同、速度重置
- 关卡过渡有短暂动画提示

### 7. 音效系统

使用 Web Audio API 生成简单音效（无需加载音频文件）。

- 吃食物音效
- 吃金苹果特殊音效
- 死亡音效
- 关卡通过音效
- 可静音切换按钮

---

## 🟢 P2 — 锦上添花

### 8. 排行榜

- 基于 localStorage 的本地排行榜（Top 10）
- 记录分数 + 日期 + 难度

### 9. 皮肤系统

- 多种蛇身配色方案（经典绿、霓虹紫、火焰橙…）
- 用通关分数解锁
- 当前选择保存到 localStorage

### 10. 双人模式

- 同键盘双人对战：Player 1 用 WASD，Player 2 用方向键
- 双方共享同一地图，可以互相碰撞致死
- 各自计分，先死的输

### 11. 道具系统扩展

在金苹果基础上，增加更多道具：

| 道具 | 效果 | 持续时间 |
|------|------|----------|
| 🍎 金苹果 | +3 分 | 5 秒后消失 |
| ⚡ 加速药水 | 速度 ×2 | 5 秒 |
| 🐢 减速药水 | 速度 ÷2 | 5 秒 |
| 🧲 磁铁 | 自动吸附最近食物 | 8 秒 |
| 👻 穿墙 | 穿过墙壁 | 8 秒 |
| ✂️ 缩短 | 蛇身减半 | 即时 |

---

## 建议实施顺序

```
P0: 金苹果 → 移动端支持 → 难度选择
P1: 音效 → 穿墙模式 → 障碍物 → 关卡
P2: 排行榜 → 皮肤 → 道具扩展 → 双人模式
```
