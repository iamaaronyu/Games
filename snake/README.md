# 🐍 贪吃蛇 — 游戏使用说明

## 快速开始

浏览器打开 `snake.html` 即可开始游戏。

```bash
open snake.html
```

## 操作方式

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` | 控制蛇的移动方向 |
| `W` `A` `S` `D` | 方向键的替代方案 |
| `Space` | 暂停 / 继续 |

## 游戏规则

- 蛇在地图上移动，吃到**红色食物**得分 +1，蛇身变长 1 格
- **撞墙** 或 **撞到自己** 则游戏结束
- 每吃一个食物，移动速度略微加快
- 最高分会自动保存到浏览器本地存储

## 游戏参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 地图大小 | 20 × 20 | 网格数 |
| 初始速度 | 200ms/格 | 约每秒移动 5 格 |
| 最快速度 | 60ms/格 | 约每秒移动 17 格 |
| 速度递增 | 5ms/食物 | 吃 28 个食物后达到最快 |

## 视觉设计

- **蛇身**：从头部亮绿色渐变到尾部暗绿色
- **蛇头**：带有白色眼睛和黑色瞳孔，跟随移动方向转动
- **食物**：红色圆形，带呼吸脉冲光晕
- **暂停**：半透明遮罩层，显示"暂停中"
- **结束**：弹出结算面板，显示最终得分
